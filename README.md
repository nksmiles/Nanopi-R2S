# R2S with USB Wifi drivers

https://github.com/coolsnowwolf/lede/

With most USB WiFi drivers included.

* TTYD terminal
* LED config
* Passwall

* jd-dailybonus
* mia- net access time control
* UnblockNeteaseMusic 
* vlmcsd
* Upnp
